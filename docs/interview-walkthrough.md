# Interview Walkthrough (Fast Path)

## 1) Start here (30 sec)
- Open root `README.md`
- Pick one infra project + one software project

## 2) Validate engineering depth (60 sec)
- Read `Architecture`
- Check CI pipeline and tests
- Check security/reliability controls (SLO, alerts, policy checks)

## 3) Validate ownership (30 sec)
- Read `Trade-offs`
- Read `What I'd improve next`

## 4) Validate execution (30 sec)
- Check tags/releases/changelog
- Check if docs match actual code and scripts
